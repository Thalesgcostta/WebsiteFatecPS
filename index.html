Processo de Desenvolvimento

Desenvolvimento da pesquisa

O questionário socioeconômico é um instrumento para coleta de informações sobre alguns aspectos da sua vida escolar, e de suas condições socioeconômicas e culturais, serve de base para as pesquisas de avaliação educacional. Foi com esse intuito que realizamos a pesquisa em prol da Fatec Dr Tomaz Novelino da cidade de Franca-SP.

Elaborar o formulário

As perguntas foram feitas para extrair informações socioeconômicas sobre o perfil dos alunos do 2º ciclo de ADS da Fatec Franca

Coletar os dados recebidos

Foi feito um formulário no Google Forms, para coletar os dados e receber as respostas, os dados formam exportados para uma planilha no Excel com o formato xlsx

Processar os dados

O processamento dos dados foi realizado por um programa em JavaScript e JSON, os dados são tratados para facilitar a construção dos gráficos

Construir os gráficos

Os gráficos são construídos de forma automatizada, pelo mesmo programa que processa os dados, utilizamos o design Google Chart

Mostrar os resultados

Os resultados podem ser vistos por esse Website, clicando nas categorias no topo do site

Escrever o relatório

O relatório foi feito para detalhar o procedimento quanto ao desenvolvimento do projeto, e para mostrar os resultados
